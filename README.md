# 🏛️ Football Referee (Legacy Edition)

Este es el origen del proyecto **Football Referee**. Representa la primera versión funcional del simulador, desarrollada enteramente con tecnologías web vainilla (HTML, CSS y JavaScript puro) sin el uso de motores gráficos en 3D ni motores de física externos.

## 💾 Estado del Proyecto
Esta rama se mantiene por valor histórico y para referencia del código original. Es ideal para aquellos que deseen aprender cómo gestionar la lógica de un juego de deportes desde cero usando Canvas y manipulación directa del DOM.

## 🛠️ Tecnologías
- **HTML5**: Estructura del juego y contenedores de UI.
- **CSS3**: Estilización de menús y diseño responsivo original.
- **JavaScript (Vanilla)**: Lógica de juego, gestión de estados de equipo y movimientos básicos.

## ⚽ Características Originales
- Sistema de equipos basado en archivos JSON.
- Lógica de juego en 2D optimizada para navegadores.
- Gestión de base de datos local sencilla.

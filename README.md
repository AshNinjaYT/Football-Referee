# ⚽ Football Referee - Original Production Source

Bienvenido a la rama principal de **Football Referee**. Este repositorio contiene la base de código original del simulador, diseñada para ser ejecutada directamente en entornos web estándar sin necesidad de compilación o motores 3D pesados.

## 🛠️ Stack Tecnológico (Core)

- **JavaScript (Vanilla)**: Gestión de la lógica de juego y comportamiento de los jugadores mediante programación procedural.
- **HTML5**: Estructura del simulador y visualización mediante Canvas 2D.
- **CSS3**: Estilización de la interfaz de usuario y menús tácticos.
- **JSON**: Almacenamiento y carga de plantillas de equipos.

## 🧠 Funcionamiento del Simulador
Esta versión utiliza una arquitectura basada en **APIs nativas del navegador**:
- **Ciclo de Animación**: Controlado mediante peticiones de frames al sistema para una visualización fluida en 2D.
- **Detección de Colisiones**: Algoritmos matemáticos directos para calcular la interacción entre el balón y los jugadores.
- **Gestión de Datos**: Carga dinámica de equipos desde archivos JSON locales.

## ⚽ Características
- **Ligereza Extrema**: Funciona en prácticamente cualquier dispositivo con un navegador moderno.
- **Código Abierto y Directo**: Estructura de archivos plana para facilitar la modificación y el aprendizaje de las bases del desarrollo de videojuegos web.
- **Independencia de Frameworks**: No requiere Node.js, Vite ni dependencias externas para su funcionamiento básico.
